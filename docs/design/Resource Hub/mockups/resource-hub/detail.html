Slate-authored · Notes for techs

Vendor torque spec on the front foot bracket reads 22 ft-lb — that's correct for the M8 bolt called out, but on builds before VIN suffix …HJ4471 the bracket ships with M10 hardware. Confirm hardware before torquing; if M10, use 32 ft-lb per the table on p. 9.

  • Vendor diagram p. 6 shows the rear crossbar 18" from the slider — current production is 22". Measure off the d-pillar, not the slider track.
  • Sealant: vendor says any butyl. Use only 3M 08509 — others have caused leaks on the 144 HR rain channel.

— J. Reyes, Slate engineering · last reviewed 2026-04-28
