# Supervisor Dashboard Claude design reference

This folder is design reference only.

Do not ship these React prototype files in the WordPress plugin.

Do not copy the prototype sidebar, topbar, tweaks panel, or app shell into production.

Production implementation must be PHP, scoped CSS, and vanilla JS.

Production route is `/ops/supervisor-dashboard`.

The deployable WordPress plugin ZIP must not include this `design/` folder, `.jsx` files, `node_modules/`, prototype-only files, or nested plugin folders.
